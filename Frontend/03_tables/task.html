Dako Workshop "Grundlagen Html & Javascript"

Aufgabe: Der Nutzer kann verschiedene Elemente auf eine Liste setzen und es wird die Anzahl der jeweiligen Elemente ausgegeben

  1. Erstelle zwei divs
    • Links für die Eingabe mit einer Textarea
    • Rechts für die Ausgabe mit einer Tabelle
  2. Füge einen Button ein, der bei einem Klick den String im Textarea invertiert
  3. Füge zwischen den beiden Textarea einen weiteren Button ein, welcher folgendes bei einem Klick tut:
    • Leeren der Ausgabetabelle
    • Auslesen der Eingabe-Textarea
    • Trennen des Strings bei einem Kommas in eine Liste von Strings
    • Befüllen der Tabelle nach folgendem Schema
      • Erste Spalte: Substring
      • Zweite Spalte: Wie oft ist der Substring im String vorgekommen
  4. Füge neben dem Button eine Checkbox ein
    • Ist die Checkbox angehakt, sollen die Zeilen nach der Anzahlspalte sortiert werden
