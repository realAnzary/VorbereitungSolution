Dako Workshop "Grundlagen Html & Javascript"

Aufgabe: Lege einen Container an, der sich durch Klick auf einen Button verschiebt.

  1. Erzeuge einen div-Container mit der Id MoveContainer mit folgenden CSS-Eigenschaften (task.css):
    • Breite (width): 50px
    • Höhe (height): 50px
    • Hintergrundfarbe (background-color): blau
    • Rand (border): rot mit Breite von 3px
    • Position: absolute
    • Position top: 0
    • Position left: 0
  2. Ändere die Hintergrundfarbe des Containers #MoveContainer mittels Javascript (task.js) auf grün und die Randfarbe auf gelb
  3. Hänge mittels addEventListener einen click-Event an den Button #ToggleMoveContainerBtn, um den Container #MoveContainer ein- bzw. auszublenden.
    1. Hänge einen click-EventListener an den Button #ToggleMoveContainerBtn
    2. Per Klick soll der Container #MoveContainer ein- bzw. ausgeblendet werden.
    3. Bei Klick soll der Value des Buttons #ToggleMoveContainerBtn Anzeigen bzw. Verstecken sein - je nachdem, ob der Container #MoveContainer derzeit sichtbar ist.
  4. Erstelle einen zweiten Button #MoveContainerBtn, der per Klick den Container #MoveContainer nach rechts bewegt:
    1. Erstelle hinter dem Button #ToggleMoveContainerBtn einen zweiten Button #MoveContainerBtn
    2. Bei jedem Klick bewegt sich der Container #MoveContainer um 100px weiter nach rechts.
    3. Kann der Container sich nicht mehr nach rechts bewegen, bewegt er sich wieder zurück nach links, usw.
  5. Erstelle eine Textbox mit Id MoveContainerSteps (input) vor dem Button #MoveContainerBtn:
    1. Value soll 100 sein
    2. Bei Klick auf den Button #MoveContainerBtn soll der Container #MoveContainer um die eingegebene Länge bewegt werden.
    3. Wird der Wert in der Textbox #MoveContainerSteps komplett gelöscht und verliert die Box den Fokus (onchange-Event), soll der Wert 100 eingetragen.
