Elemente erstellen und verschieben

Aufgabe: Erstelle eine Liste und verschiebe die Einträge

  1. Hänge an dem Button #AddAnimalBtn ein click-EventListener mit folgenden Ausführungen (task.js):
    • Im Div #Fence1 ein neues div-Element mit der Klasse Animal anlegen
    • Der Container .Animal enthält ein span-Element mit dem Text aus der Textbox #AddAnimal
    • Verhindere ein Hinzufügen, wenn es bereits ein Element .Animal mit gleichem Namen gibt.
  2. Style das Element .Animal (task.css):
    1. Innenabstand: 5px
    2. Außenabstand: 3px
    3. Schriftfarbe: darkgreen
    4. Hintergrundfarbe: hellgrau
  3. Hänge beim Anlegen eines Containers .Animal mittels addEventListener einen click-Event, in dem die Klasse Selected an den Container .Animal hinzugefügt bzw. entfernt wird.
  4. Style alle Elemente .Animal mit der Klasse Selected:
    1. Hintergrundfarbe: lightslategrey
    2. Schriftfarbe: weiß
  5. Hänge mittels addEventListener einen click-Event an den Button #MoveAnimalsBtn, um alle Elemente .Animals mit der Klasse Selected in den jeweils anderen div-Container mit der Klasse .Fence zu verschieben.
  6. Erweitere den Click-EventListener von #AddAnimalBtn:
    1. Überprüfe, ob der eingegebene Name aus der Textbox #AddAnimal bereits in beiden Elementen .Fence besteht.
    2. Füge das neue Element .Animal alphabetisch sortiert in den Container #Fence1 ein.
  7. Erweitere den Click-EventListener von #MoveAnimalsBtn: Verschiebe die selektieren Container .Animal und füge diese alphabetisch sortiert in den jeweiligen Container .Fence ein.
  8. Ermögliche das Verschieben von .Animal per drag and drop in den jeweils anderen Container .Fence.
